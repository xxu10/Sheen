# sheen
Lightweight Koa web framework.

# Installation
npm install sheen10

# Catalog

- config //config files
    - default.js    
- controller    //controller (router)
    - index.js  
- schedule  //scheduling tasks
    - test.js  
- models //database
    - index.js  
- public // web resources
- views  // html templates
- lib // core of the web framework
    - controller.js
    - db.js
    - index.js
    - log.js
    - main.js
    - schedule.js
    - utils.js
- app.js    //init file
- index.js  //entrance to the project
